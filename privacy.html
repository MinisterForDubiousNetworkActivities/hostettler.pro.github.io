Privacy Policy

Last updated: 16/06/2026

This page explains what little data is processed when you visit hostettler.pro, in accordance with the General Data Protection Regulation (GDPR / DSGVO). This is a static website: there is no JavaScript, no analytics, no tracking pixels, no contact form, and no user accounts.

1. Controller

Lennart Hostettler
Email: lennart.hostettler@proton.me

2. Hosting (GitHub Pages)

This site is hosted on GitHub Pages, a service provided by GitHub, Inc. / GitHub B.V. (a Microsoft company). When you visit this site, GitHub's servers process technical data necessary to deliver the page to you — typically your IP address, browser/user-agent string, referring URL, the page requested, and a timestamp. This happens automatically for every request to any website, regardless of whether it has a privacy policy, and is not something this site controls or can opt out of on your behalf.

This processing is necessary for the operation and security of the service (legitimate interest, Art. 6(1)(f) GDPR). GitHub is based in the United States; where data is processed there, GitHub relies on appropriate safeguards (such as Standard Contractual Clauses and/or its participation in the EU–U.S. Data Privacy Framework) as described in its own policy. See GitHub's Privacy Statement for details on retention periods and GitHub's role as a processor.

3. DNS (Cloudflare)

The domain hostettler.pro uses Cloudflare for DNS resolution only (no proxying/CDN). This means Cloudflare's nameservers see which domain a DNS resolver is asking about when your device looks up hostettler.pro, but Cloudflare does not sit between you and GitHub Pages and does not receive the actual page content or your requests to the site. See Cloudflare's Privacy Policy for how it handles DNS query data.

4. Cookies & Tracking

This site does not set any cookies, does not use any analytics or tracking scripts, and does not embed third-party fonts, ads, or widgets. No consent banner is shown because none is required — there is nothing non-essential to consent to.

5. Contact via Email

The "Contact" link in the footer is a plain mailto: link. Clicking it opens your own email client; no message content passes through or is stored by this website. Whatever you choose to send is handled according to your own and my email provider's policies.

6. External Links

Pages on this site link out to third-party services (e.g. GitHub, LinkedIn, vendor repositories, other blogs). Once you follow such a link, that site's own privacy policy applies — this policy only covers hostettler.pro itself.
